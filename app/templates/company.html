This is a personal / demo project, not a registered company. This page describes the project and its goals.

What this is

{{ app_name }} is a self-hosted dashboard for tracking a home solar installation — generation, consumption, grid import/export, battery health, billing, and maintenance alerts, all in one place.

Why it exists

Most inverter apps lock your data behind a vendor's cloud. This project takes the opposite approach: your readings live in your own PostgreSQL database, your login is your own, and the data source is swappable — start with a simulator, move to a real inverter over Modbus when you're ready.

How it's built

Python backend (FastAPI), PostgreSQL for storage, a background scheduler for live readings, and a server-rendered dashboard with Chart.js for the trend view. No third-party analytics, no vendor lock-in.
